# VRM 动画资源下载指南

## 推荐的动画资源网站

### 1. **VRoid Hub** (官方推荐)
- **网址**: https://hub.vroid.com/
- **说明**: VRoid 官方模型和动画分享平台
- **特点**: 
  - 免费和付费动画都有
  - 支持 VRM 格式
  - 可以直接预览动画效果
- **搜索**: 在搜索框输入 "animation" 或 "モーション"

### 2. **Booth** (日本最大的数字内容市场)
- **网址**: https://booth.pm/
- **说明**: 日本最大的数字内容市场，有大量 VRM 相关资源
- **特点**:
  - 大量免费和付费动画包
  - 支持 VRM/VRoid 动画
  - 价格从免费到几千日元不等
- **搜索关键词**: 
  - "VRM モーション" (VRM 动作)
  - "VRoid アニメーション" (VRoid 动画)
  - "表情アニメーション" (表情动画)

### 4. **GitHub 开源动画**
- **搜索**: GitHub 上搜索 "VRM animation" 或 "VRM motion"
- **格式**: 通常是 `.vrm` 文件或 `.gltf` 文件
- **说明**: 一些开发者会开源他们的动画资源

## 动画文件格式

VRM 动画通常使用以下格式：

1. **VRM 文件** (`.vrm`)
   - 包含模型和动画的完整文件
   - 可以直接加载

2. **GLTF/GLB 文件** (`.gltf` / `.glb`)
   - 标准 3D 格式
   - Three.js 原生支持
   - 可以包含动画数据

3. **VRM 动画数据** (JSON)
   - 一些工具导出的动画数据
   - 需要手动集成

## 注意事项

1. **版权**: 下载前请确认动画的使用许可
2. **格式兼容性**: 确保动画文件与你的 VRM 模型兼容
3. **骨骼映射**: 某些动画可能需要特定的骨骼结构
4. **文件大小**: 动画文件可能较大，注意加载时间

## 快速开始

1. 访问 https://booth.pm/
2. 搜索 "VRM モーション" 或 "VRM アニメーション"
3. 选择免费或付费的动画包
4. 下载 `.glb` 或 `.gltf` 格式的动画文件
5. 将文件放到 `static/models/vrm/animations/` 目录
6. 在代码中加载并播放动画
